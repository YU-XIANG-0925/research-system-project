# 演講能力輔助系統的資料模型

## 實體

### 使用者 (User)
- **描述**：系統的使用者。
- **欄位**：
    - `id`：字串 (唯一識別碼)
    - `name`：字串
    - `email`：字串

### 講稿 (Script)
- **描述**：使用者上傳的演講稿。
- **欄位**：
    - `id`：字串 (唯一識別碼)
    - `user_id`：字串 (指向使用者的外鍵)
    - `title`：字串
    - `content`：字串 (演講的全文內容)
    - `annotations`：標註物件列表

### 標註 (Annotation)
- **描述**：在講稿中特定位置的手勢或情緒建議。
- **欄位**：
    - `id`：字串 (唯一識別碼)
    - `type`：字串 ("gesture" 或 "emotion")
    - `value`：字串 (例如："舉手"、"微笑")
    - `start_index`：整數 (在講稿內容中的起始字元索引)
    - `end_index`：整數 (在講稿內容中的結束字元索引)

### 自定義動作 (CustomAction)
- **描述**：使用者定義的機器人動作。
- **欄位**：
    - `id`：字串 (唯一識別碼)
    - `user_id`：字串 (指向使用者的外鍵)
    - `name`：字串
    - `mqtt_command`：字串 (要發送給機器人的命令)

### 硬體設定 (HardwareConfiguration)
- **描述**：外部硬體的連線設定。
- **欄位**：
    - `id`：字串 (唯一識別碼)
    - `user_id`：字串 (指向使用者的外鍵)
    - `broker_address`：字串
    - `broker_port`：整數
    - `topic`：字串
