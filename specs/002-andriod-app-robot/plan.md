## 技術背景

這個機器人專案有幾個重要的點必須遵守：
1.  專案的建立必須嚴格參考 "https://github.com/YU-XIANG-0925/NuwaSDKExample.git" 這個 repo在 origin/main 所提供的範例。
2.  如果 "https://github.com/YU-XIANG-0925/NuwaSDKExample.git" origin/main 無法使用，就改為嚴格參考samples/NuwaSDKExample 所提供的範例。
3.  需要使用 Java 程式語言開發，並使用 MQTT 相關函式庫。
4.  專用的 SDK 被放置在 `tools` 的資料夾中，建立專案時必須將 SDK 檔案複製到正確的位置 `app/libs/`。
